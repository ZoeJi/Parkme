# ParkMe #
This is a free parking finder mobile application.

### Technology used:
react/react-native, express.js, mongoDB

### App explained:
This app applies the crowdsourcing idea. 
Users can quickly find a free parking location near them, and add free parking location when they find any. In addition, users are able to report any expired or changed free parking location so the administrator can remove it.
